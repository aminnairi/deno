# Contributing

## Requirements

- [Git](https://git-scm.com/)
- [Docker](https://www.docker.com/)

## Installation

```console
$ git clone https://github.com/aminnairi/deno.git
$ cd deno
```

## Update

Update the [`Dockerfile`](./docker/Dockerfile) file.

## Install

```console
$ make install
```

## Uninstall

```console
$ make uninstall
```
